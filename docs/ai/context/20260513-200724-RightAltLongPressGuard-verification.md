## 当前实现范围

已实现：

1. `RightAlt` 长按超过 `500ms` 时不进入识别。
2. 长按会显示提示弹窗。
3. 弹窗可通过右上角关闭。
4. 弹窗显示期间再次点按 `RightAlt` 会先关闭弹窗，再允许新的按键周期继续识别。
5. `Dashboard` 页面级快捷键监听已移除，统一收口到 `AppShell`。

## 仍需人工交互确认

下面这些行为需要在桌面窗口里手测：

1. 短按 `RightAlt`
   - 应正常触发识别胶囊
2. 长按 `RightAlt` 超过 `500ms`
   - 不应进入识别
   - 应弹出长按提示
3. 长按后松开
   - 提示仍保留
4. 点击 `×`
   - 提示关闭
5. 提示显示后再次短按 `RightAlt`
   - 应先关闭提示
   - 再正常触发识别
6. 提示显示后再次长按 `RightAlt`
   - 应先关闭旧提示
   - 再重新弹出新提示
